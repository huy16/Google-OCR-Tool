# Hướng dẫn sử dụng Web App Geocoding

## 1. Khởi động
1. Mở thư mục `d:\TOOL GOOGLE ANTIGRAVITY\7. Tool OCR Google Map\Web_App`
2. Chuột phải chọn "Open Terminal" hoặc gõ `cmd` trên thanh địa chỉ.
3. Chạy lệnh:
   ```bash
   node server.js
   ```
4. Mở trình duyệt truy cập: [http://localhost:3000](http://localhost:3000)

## 2. Sử dụng
- Upload file Excel đầu vào.
- Click "Start Geocoding".
- Theo dõi tiến trình ngay trên web.
- Tải file kết quả về khi hoàn tất.

## 3. Lưu ý
- Đừng tắt cửa sổ đen (Terminal) khi web đang chạy.
- Chrome sẽ bật lên để xử lý (chế độ ẩn hoặc hiện tùy cấu hình), đừng tắt nó.
